# 回归模型报告：腰围预测

## 模型概览

本项目构建了4个回归模型用于预测腰围：
- 线性回归（弹性网络）
- 随机森林回归
- XGBoost回归
- 支持向量机回归（SVM）

|------|--------------|-------------|-----|----------|

## 最佳模型

**随机森林**
- RMSE: 5.87 cm

- MAPE: 4.96%

## 模型解读

### 预测能力
R² = 0.9339 表示模型能解释 93.4% 的腰围变异。

该模型可用于：
